## 代码演示

### 警告通知

```jsx
import { Notification } from 'tuya-panel-kit'

Notification.show({
  message: '警告提示框',
  onClose: () => {
    Notification.hide();
  },
  theme: {
    successIcon: 'red',
    errorIcon: 'yellow',
    warningIcon: 'black',
  },
});
```
