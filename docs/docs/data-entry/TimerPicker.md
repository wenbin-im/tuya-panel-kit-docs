## 代码演示

### 时间段选择器基础使用

```jsx
import { TimerPicker } from 'tuya-panel-kit'

const [time, setTime] = React.useState({ startTime: 0, endTime: 0 });

<TimerPicker
  style={{ marginTop: 10 }}
  startTime={time.startTime}
  endTime={time.endTime}
  onTimerChange={(startTime, endTime) => setTime({ startTime, endTime })}
/>
```

### 时间段选择器配合前缀位置

```jsx
import { TimerPicker } from 'tuya-panel-kit'

const [time, setTime] = React.useState({ startTime: 0, endTime: 0 });

<TimerPicker
  style={{ marginTop: 10 }}
  startTime={time.startTime}
  endTime={time.endTime}
  prefixPosition="left"
  onTimerChange={(startTime, endTime) => setTime({ startTime, endTime })}
/>
```
